List of JayData files:

JayData.js					- JayData library including customized jslint.
JayData.min.js				- Minified JayData library including customized jslint.
JayData-vsdoc.js			- JayData library including customized jslint, used only in Visual Studio 2010 development time.

/jaydatamodules				- JayData modules folder
deferred.js					- Support for jQuery deferred object
errorhandler.js				- Dummy error handler module for development
formBinder.js				- Formbinder module
inMemory.js					- toQueryable extension for in memory collections
knockout.js					- Support for knockoutJS
sencha.js					- Support for Sencha
template.js					- Support for templating
validate.js					- Form validation module

/jaydataproviders			- JayData providers (minified versions also present)
FacebookProvider.js			- Facebook provider
IndexedDbProvider.js		- IndexedDb provider
InMemoryProvider.js			- InMemory provider
MongoDbProvider.js			- MongoDb provider
oDataProvider.js			- oDataProvider.js
SqLiteProvider.js			- sqLite provider
StormProvider.js			- JayStorm provider
YQLProvider					- YQL (Yahoo Query Language) provider
